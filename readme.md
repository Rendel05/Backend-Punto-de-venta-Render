Backend - Sistema Punto de Venta (POS)Descripción

Este proyecto corresponde al backend de un sistema de Punto de Venta (POS).
Se encarga de gestionar la lógica de negocio, autenticación de usuarios, manejo de productos, ventas y comunicación con la base de datos.

El sistema está estructurado siguiendo buenas prácticas de organización y separación de responsabilidades para facilitar su mantenimiento y escalabilidad.

Objetivos del Proyecto

Implementar la lógica central del sistema POS.

Gestionar usuarios y roles.

Administrar productos e inventario.

Registrar ventas.

Conectar con base de datos de forma segura.

Mantener una arquitectura organizada y modular.

Características

Autenticación de usuarios

Gestión de roles (Administrador / Usuario)

CRUD de productos

Registro de ventas

Conexión a base de datos

Configuración mediante variables de entorno (.env)
